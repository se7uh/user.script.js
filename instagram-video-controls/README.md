# Instagram Video Controls

A userscript that provides enhanced video playback experience on Instagram with floating controls and multiple speed options.

## Preview

![Instagram Video Controls Preview](https://github.com/user-attachments/assets/8590171d-0222-44dd-b43a-af41b54e5cc3)

*Custom play/pause controls, seek bar, and speed selector that appear on hover*

## Features

- **Auto-Hiding Controls** - Clean interface that appears on hover and disappears when not needed
- **Play/Pause Button** - Toggle playback with intuitive SVG icons
- **Seek Bar** - Scrub through video content with precision
- **Speed Selection** - Choose from multiple playback speeds (0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x)

## Installation

Install this script from [Greasy Fork](https://greasyfork.org/en/scripts/instagram-video-controls)
